** Accessing Datafiles**
1) Use psiturk command
2) Use "download_datafiles" command
3) Exit to the main astrobee folder
4) Use the "psiturk_formatter.py" script
5) Data is now saved "minimally_formatted_questiondata.csv"

** PsiTurk Conditions**
Condition = 0, 4,  8, or 12: associative building
          = 1, 5,  9, or 13: associative damaging
	  = 2, 6, 10, or 14: dissociative building
	  = 3, 7, 11, or 15: dissociative damaging
